操作说明
APP界面操作

触摸屏幕左或右边部分为弹出频道列表
TV设备遥控按键OK或Enter键弹出频道列表
触摸屏幕中间部分为弹出底部信息栏
长按屏幕右部分为弹出设置列表，长按屏幕左部分为弹出搜索界面
屏幕右上下滑动为切换频道，左部分上下滑动为音量调整
屏幕左右滑动为切换线路（回看或视频片段时弹出进度条界面）
列表界面操作

触摸屏长按单个节目添加收藏/取消收藏
TV设备遥控按菜单键或长按OK按键收藏/取消收藏
订阅界面操作

地址列表中，长按地址可进入编辑状态
点击可切换地址，编辑状态下按删除按钮或确定按钮为完成编辑
触摸二维码可跳转至远程页面
媒体信息操作

长按对应信息可复制对应内容
配置说明
解码方式

支持系统、ijkplayer及exoplayer，不同设备切换合适的解码
根据记忆解码开关，开启为单独设置，关闭为全局生效
分类中可指定某个分组的解码方式（如：DE=0为系统解码），配置后设置中更换解码不会生效
EPG配置

需选择解析方式，默认DIYP解析（推荐使用XmlTv格式EPG地址和解析）
EPG地址需配置对应的解析方式，否则不能加载
EPG订阅栏可配置EPG的时间偏移值以及缓存更新时间
EPG缓存

EPG地址配置后选择对应的解析方式，加载后会自动缓存（可关闭）
缓存自动清除6天前的，（非xml文件）明天后天及精彩节目epg不缓存
EPG数据凌晨后启动会自动更新一次，根据缓存设置时间会进行二次更新

脚本说明
频道地址格式

格式1

//js脚本
http://xx/ku9/js/xx.xx?id=xx

//py脚本
http://xx/ku9/py/xx.xx?id=xx
字符/ku9/js/或/ku9/py/为识别格式，xx为任意字符

格式2（网络路径）

//js脚本
http://xx/xx/k-web/ku9/js/xx.xx?id=xx

//py脚本
http://xx/xx/k-web/ku9/py/xx.xx?id=xx
/k-web/前为任意网络路径

格式3（时移地址）

//js脚本
http://xx/xx/k-web/PLTV/ku9/js/xx.xx?id=xx

//py脚本
http://xx/xx/k-web/PLTV/ku9/py/xx.xx?id=xx
在/k-Web/之后添加可时移字符，如PLTV

格式4（指定json文件,获取url标签，类似重定向地址）

http://xx/xx/xx.xx?id=xx&/ku9-json/
http://xx/xx/xx.xx为具体json路径地址

url同时包含/k-web/和/ku9/js/或/ku9/py/读取网络文件，只包含/ku9/js/或/ku9/py/读取本地文件； py仅py版本支持

结合js脚本使用
function main(item) {

    //以PLTV作为示例，添加PLTV让播放器识别为可回看，不同源返回回看格式不同，这里通过播放器返回的playseek可以自定义最终的回看格式
    //源格式示例：源1,http://PLTV/ku9/js/**.js?liveUrl=http://xxx0001&backUrl=rtsp://xxx0002

    let url = item.url;

    //获取直播链接 http://xxx0001
    const liveUrl = ku9.getQuery( url, "liveUrl" );

    //获取取回看链接 rtsp://xxx0002
    const backUrl = ku9.getQuery( url, "backUrl" );

    //获取PLTV源回看时间（yyyyMMddHHmmss格式时间段）
    const playSeek = ku9.getQuery( url, "playseek" );

    if (!playSeek) {
        //设置live时的解码方式、画面比例、exo bufferForPlaybackMs值、headers
        return { url: liveUrl, player: 1, scale: 0 , exobufms: 2500 , headers: { 'User-Agent': 'okhttp/3.12.11' } };
    } else {
        //设置back时的解码方式、画面比例、ijk AnalyzeDuration值、host
        let finalUrl = backUrl + "?playseek=" + playSeek; //需要定义的回看格式
        return { url: finalUrl, player: 3, scale: 2 , ijkad: 2000000 , host: "127.0.0.1" };
    }
}

versionCode值：apk文件AndroidManifest.xml中的versionCode值

酷9默认versionCode为17，update.json文件中的versionCode为1，所以默认直接使用不会提示升级
